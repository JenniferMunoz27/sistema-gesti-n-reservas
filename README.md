# Sistema Integral de Gestión de Clientes, Servicios y Reservas

## Descripción
Este proyecto corresponde al desarrollo de un sistema en Python orientado a objetos para la gestión de clientes, servicios y reservas en una empresa ficticia llamada Software FJ.

El sistema fue desarrollado aplicando principios fundamentales de la programación orientada a objetos como la encapsulación, herencia, polimorfismo y abstracción.

## Objetivo
Desarrollar una aplicación funcional que permita gestionar información sin el uso de bases de datos, implementando manejo de excepciones para garantizar la estabilidad del sistema.

## Funcionalidades
- Registro de clientes
- Gestión de servicios
- Creación de reservas
- Validación de datos
- Manejo de errores
- Registro de logs

## Tecnologías utilizadas
- Python

## Ejecución del programa
Para ejecutar el sistema, usar el siguiente comando:

python proyecto.py
